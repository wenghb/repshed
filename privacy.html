Privacy Policy

Last updated: March 19, 2026

🔒 TL;DR — We collect nothing. Your audio files never leave your device. There are no accounts, no cookies, no analytics, no tracking.

1. What Data We Collect

None. RepShed does not collect, store, transmit, or process any personal data. Specifically:

  • No user accounts or registration
  • No cookies or local storage tracking
  • No analytics scripts (no Google Analytics, no Mixpanel, nothing)
  • No advertising or ad tracking
  • No server-side processing of any kind

2. Audio Files

When you load an audio file into RepShed, the file is processed entirely within your web browser using the Web Audio API. Your audio files are:

  • Never uploaded to any server
  • Never transmitted over the internet
  • Never stored persistently (they exist only in browser memory while the page is open)
  • Completely discarded when you close or refresh the page

3. Hosting & Infrastructure

RepShed is hosted on Cloudflare Pages as a static website. Cloudflare may collect basic server logs (IP address, browser user agent, timestamp) as part of standard web hosting operations. This is governed by Cloudflare's Privacy Policy. We do not access or use these logs.

4. External Links

RepShed contains a link to Stripe for optional donations. If you click this link, you leave RepShed and are subject to Stripe's privacy policy. We have no control over their data practices.

5. Children's Privacy

RepShed does not collect any data from anyone, including children. There is no age restriction to use RepShed.

6. Changes to This Policy

If we ever change how RepShed handles data (which would only happen if we add features that require it), we will update this page with a new "Last updated" date. Since RepShed currently collects zero data, any change would be clearly noticeable.

7. Contact

If you have questions about this privacy policy, you can reach us at support@repshed.com.
